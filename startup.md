# Startup Guide

Follow these instructions to get the Event Management App running on your local machine.

## Prerequisites
1. **Node.js**: Ensure you have Node.js installed. (Check with `node -v`)

## 1. Installation
Open your terminal in the project directory and install the required npm packages:

```bash
npm install
```

## 2. Running the App
Start the Express server by running:
```bash
npm start
```
You should see a message in your terminal confirming the server is running:
```text
Server running at http://localhost:3000
```

## 3. Usage
Open your web browser and navigate to:
[http://localhost:3000](http://localhost:3000)

You can now register a test user, log in, and start creating events! The data will be saved locally to `backend/data/db.json`.
